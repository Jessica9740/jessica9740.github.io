Mac 초기화 이후 최초 설정 방법 완벽 가이드

Mac을 새로 구매했거나 초기화한 후, 어떤 순서로 설정을 진행해야 가장 효율적일까요? 단순히 사용만 하는 분부터 개발자까지 모두 만족할 수 있는 단계별 셋업 가이드를 준비했습니다. 이 글을 그대로 따라가면 30분 내에 완벽한 작업 환경을 구축할 수 있습니다.

1단계: macOS 초기 설정 마법사 진행

Mac의 전원을 처음 켜면 설정 마법사가 자동으로 실행됩니다. 다음 항목들을 순서대로 진행하세요.

  • 국가 및 언어 선택: 대한민국, 한국어 선택
  • Wi-Fi 연결: 네트워크에 연결
  • Apple ID 로그인: iCloud, App Store 사용을 위해 필수
  • Touch ID 등록: 보안과 편의성을 위해 권장
  • Siri 설정: 필요에 따라 선택

💡 TIP: 이전 Mac에서 데이터를 옮기려면 "마이그레이션 어시스턴트"를 사용하세요. 단, 깨끗한 환경을 원한다면 새로 설정하기를 추천합니다.

2단계: 시스템 환경설정 최적화

트랙패드 & 마우스 설정

시스템 설정 > 트랙패드에서 다음 항목을 활성화하세요.

  • 탭하여 클릭: 살짝 두드리기만 해도 클릭
  • 궤적 속도: 가장 빠르게 설정하면 작업 효율 ↑
  • 세 손가락 끌기: 손쉬운 사용 > 포인터 제어 > 트랙패드 옵션에서 활성화

Dock 설정

시스템 설정 > 데스크탑 및 Dock
  • Dock 자동 가리기: 활성화 (화면 공간 확보)
  • 크기: 작게 조정
  • 최근 사용한 앱 표시: 비활성화 (깔끔함 유지)

Finder 설정

Finder를 열고 Cmd + ,로 환경설정에 진입한 후:

  • 모든 파일 확장자 보기: 활성화
  • 상태 표시줄, 경로 표시줄 보기: 보기 > 상태 표시줄 보기

숨김 파일 표시는 단축키로 토글할 수 있습니다.

Cmd + Shift + . (마침표)

3단계: 필수 시스템 보안 설정

보안은 가장 먼저 챙겨야 할 항목입니다.

# FileVault 디스크 암호화 상태 확인
fdesetup status

# 방화벽 상태 확인
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --getglobalstate

시스템 설정 > 개인정보 보호 및 보안에서:

  1. FileVault 활성화 (디스크 암호화)
  2. 방화벽 켜기
  3. 잠금 화면 설정: 5분 후 자동 잠금

4단계: Homebrew 설치 (개발자 필수)

Homebrew는 macOS의 사실상 표준 패키지 매니저입니다. 터미널을 열고 다음 명령어를 실행하세요.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

설치가 완료되면 PATH 설정이 필요합니다. Apple Silicon(M1/M2/M3) 사용자는 다음을 실행하세요.

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"

설치 확인:

brew --version
brew doctor

6단계: 터미널 환경 꾸미기 (Zsh + Oh My Zsh)

macOS 기본 셸은 zsh입니다. Oh My Zsh를 설치하면 강력한 기능과 테마를 사용할 수 있습니다.

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

유용한 플러그인 설치

# 자동 완성 플러그인
git clone https://github.com/zsh-users/zsh-autosuggestions \
  ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

# 문법 강조 플러그인
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git \
  ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

~/.zshrc 파일을 열어 플러그인을 활성화하세요.

plugins=(
  git
  zsh-autosuggestions
  zsh-syntax-highlighting
)

설정 적용:

source ~/.zshrc

7단계: Git 설정

개발자라면 Git 글로벌 설정을 빼놓을 수 없죠.

# 사용자 정보 설정
git config --global user.name "본인이름"
git config --global user.email "your-email@example.com"

# 기본 브랜치명 설정
git config --global init.defaultBranch main

# 줄바꿈 자동 변환 (macOS)
git config --global core.autocrlf input

# 설정 확인
git config --global --list

SSH 키 생성 및 GitHub 등록

# SSH 키 생성
ssh-keygen -t ed25519 -C "your-email@example.com"

# 공개 키 클립보드에 복사
pbcopy < ~/.ssh/id_ed25519.pub

복사된 키를 GitHub Settings > SSH and GPG keys에 등록하세요.

8단계: Node.js 버전 관리 (nvm)

여러 프로젝트를 다룬다면 Node.js 버전 관리가 필수입니다.

brew install nvm

# 설정 디렉토리 생성
mkdir ~/.nvm

# zshrc에 추가
cat << 'EOF' >> ~/.zshrc
export NVM_DIR="$HOME/.nvm"
[ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && \. "/opt/homebrew/opt/nvm/nvm.sh"
[ -s "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm" ] && \. "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"
EOF

source ~/.zshrc

# 최신 LTS 버전 설치
nvm install --lts
nvm use --lts

9단계: macOS 숨겨진 설정 (defaults 명령어)

GUI에서는 보이지 않는 강력한 설정들을 활성화할 수 있습니다.

# 스크린샷 저장 위치 변경
mkdir -p ~/Pictures/Screenshots
defaults write com.apple.screencapture location ~/Pictures/Screenshots

# 스크린샷 그림자 제거
defaults write com.apple.screencapture disable-shadow -bool true

# Dock 애니메이션 속도 높이기
defaults write com.apple.dock autohide-delay -float 0
defaults write com.apple.dock autohide-time-modifier -float 0.3

# 키 반복 속도 빠르게
defaults write NSGlobalDomain KeyRepeat -int 2
defaults write NSGlobalDomain InitialKeyRepeat -int 15

# 변경사항 적용
killall Dock
killall SystemUIServer

10단계: 백업 설정 (Time Machine)

마지막으로 가장 중요한 것, 백업입니다.

  1. 외장 드라이브 연결
  2. 시스템 설정 > 일반 > Time Machine 진입
  3. 백업 디스크 선택
  4. 자동 백업 활성화

마무리: 체크리스트

설정이 모두 끝났다면 다음을 확인하세요.

  • ✅ Apple ID 로그인 및 iCloud 동기화 확인
  • ✅ FileVault 암호화 활성화
  • ✅ Time Machine 백업 설정
  • ✅ Homebrew 정상 작동
  • ✅ Git 사용자 정보 설정 완료
  • ✅ 자주 쓰는 앱 설치 완료

이제 여러분의 Mac은 개발과 일반 작업 모두에 최적화된 환경이 되었습니다. 처음에는 설정할 게 많아 보이지만, 한 번 제대로 셋업해두면 앞으로 몇 년간 쾌적하게 사용할 수 있습니다.

특히 Homebrew와 dotfiles(.zshrc 등)는 GitHub에 백업해두면 다음 초기화 시 단 몇 분 만에 동일한 환경을 복구할 수 있으니 꼭 활용해보세요!
